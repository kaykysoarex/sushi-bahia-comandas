# Sushi Bahia — Painel de Pedidos

Painel web para gerenciamento de pedidos de restaurante, com visão de garçom e cozinha.

## Funcionalidades

- **Garçom:** seleciona a mesa, monta o pedido pelo cardápio e envia para a cozinha
- **Cozinha:** visualiza os pedidos em tempo real e marca como prontos
- Status das mesas atualizado automaticamente (livre → ocupada → aguardando)

## Estrutura

```
sushi-bahia-comanda/
├── index.html   — estrutura HTML
├── style.css    — estilos
└── app.js       — lógica da aplicação
```

## Como usar

Abra o `index.html` diretamente no navegador — não precisa de servidor ou instalação.

## Tecnologias

- HTML5, CSS3 e JavaScript puro (sem frameworks)
- Fontes: [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) e [Manrope](https://fonts.google.com/specimen/Manrope) via Google Fonts
